# Game Design

## Mission promise

The participant receives functional web shooters immediately after calibration.
The mission never stops for a menu, load screen, death screen, or separate level.
Chase momentum carries directly into the Void Regent reveal and containment.

## Chase beats

The chase contains ten authored, non-overlapping beats: billboard dodge, Glider Raider
capture, vent jump, barrier pull, scaffold crouch, anchor swing, civilian rescue,
crane dodge, invisible shockwave shield and two-web route stabilization. Each beat
has a large Spider-Sense prompt, a visual set piece, a response window, scoring,
and fail-soft damage.

## The Void Regent

The Void Regent is an original cosmic armored villain with a crowned helmet,
heavy pauldrons, glowing core, shader-driven motion, wire-like damage bands and
rain displacement. Correct defensive responses expose it and open web counters.
Free-fire shots require the averaged both-hand reticle to be on target. A single
debris catch-pull-release produces the cinematic sling.

## Finisher

At 78 seconds a web cage forms. Both webs and physical pull build tension.
Prompts progress through forward hands, fire, pull, more tension and final pull.
After a reasonable attempt, assistance ramps up. Containment is guaranteed by
82.4 seconds so results always begin on time.

## Fail-soft rules

Damage cannot end the fixed timeline. At zero energy, Last Chance Mode restores 25
energy, adds assistance and applies a score penalty. The third obstacle collision
disqualifies the score and produces a failed result, but the authored boss and
assisted ending still complete before automatic reset. Assistance increases after
misses and reduces after success without creating impossible patterns.

## Scoring

Points reward challenge clears, combo, web interactions, rescues, boss counters,
debris sling and final tension. Results report score, Spider-Sense, web accuracy,
rescues, perfect dodges, boss control and tension. The local leaderboard is optional
and never blocks results.
